# Cisco Networking Labs (Packet Tracer)

This repository contains a collection of **Cisco Packet Tracer (.pkt) lab files** created for learning, practicing, and demonstrating core **computer networking and Cisco routing & switching concepts**.

These labs are suitable for:
- Academic coursework (B.Tech / M.Tech)
- Cisco certification preparation (CCNA-level)
- Hands-on understanding of network behavior

---

##  Repository Structure

Each `.pkt` file represents an **independent networking lab** focusing on a specific concept such as routing, switching, security, or access control.

The labs are designed to be opened and simulated using **Cisco Packet Tracer**.

---

##  Lab Descriptions

###  `networktopology.pkt`
- Basic network topology with multiple devices
- Demonstrates:
  - IP addressing
  - Device interconnection
  - End-to-end connectivity testing

---

###  `vlan.pkt`
- Virtual LAN configuration
- Demonstrates:

  - Access and trunk ports
  - Inter-VLAN communication (if configured)

---

###  `acl.pkt`
- Access Control List (ACL) configuration
- Demonstrates:
  - Standard and/or extended ACLs
  - Traffic filtering
  - Network security using rules

---

###  `security.pkt`
- General network security configurations
- Demonstrates:
  - Traffic control
  - Device-level security concepts
  - Basic protection mechanisms

---

###  `CBAC.pkt`
- Context-Based Access Control
- Demonstrates:
  - Stateful firewall behavior
  - Inspection of traffic sessions
  - Enhanced security compared to static ACLs

---

###  `AAA.pkt`
- Authentication, Authorization, and Accounting
- Demonstrates:
  - User authentication mechanisms
  - Controlled device access
  - Security management concepts
